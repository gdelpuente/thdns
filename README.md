Thdns
===
![alt text](https://raw.githubusercontent.com/gdelpuente/thdns/master/DDNSTophost.png)

DNS dinamico con TopHost


How to run
---

configure

```
userid = "username"
passwd = "password"
dyn=['@','www']
```
```
git clone https://github.com/gdelpuente/thdns
cd thdns
python thdns.py -g
```

print help

```

python thdns.py -h
```
